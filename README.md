# node-guestbook

A simple guestbook web app built using NodeJS, Express and integrating with a MongoDB backend.

## Installation
```bash
$ git clone https://github.com/mcltn/node-guestbook
$ cd node-guestbook
$ npm install
```

## Running the server
```bash
$ export PORT=5001
$ export DATABASE_URL="mongodb://localhost:27017/guestbook"
$ npm start
```
